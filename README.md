# CISM
Welcome to the CISM Guide repository.
A comprehensive, professionally structured set of CISM study notes focused on Information Security Governance, Risk Management, Security Program Development, and Incident Management. Designed as a practical GRC reference for governance frameworks, ISO 27001/NIST alignment, metrics, policies, and real-world security management practices.
