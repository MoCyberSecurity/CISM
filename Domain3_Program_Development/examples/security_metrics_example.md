# Security Metrics Example

**Purpose:** Measure the effectiveness of the information security program.

**Types of Metrics:**
- **KRIs (Key Risk Indicators):** Number of high-risk vulnerabilities open for >30 days
- **KGIs (Key Goal Indicators):** % of systems compliant with baseline configuration
- **KPIs (Key Performance Indicators):** Mean time to detect and respond to incidents

**Notes:** Metrics should be SMART (Specific, Measurable, Attainable, Relevant, Timely) and reported to leadership regularly.
